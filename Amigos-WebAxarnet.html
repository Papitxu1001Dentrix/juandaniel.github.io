Ficha Especial: Primera Página Web del Mundo

La primera página web fue publicada el 6 de agosto de 1991 por el científico británico Tim Berners-Lee desde el CERN, la Organización Europea para la Investigación Nuclear. Esta página marcó el inicio de la World Wide Web como servicio público.

Organización responsable

  • Entidad: CERN (European Organization for Nuclear Research)
  • Ubicación: Ginebra, Suiza
  • Propósito: Facilitar el intercambio de información entre científicos mediante hipertexto

Contenido original

  • Descripción del proyecto World Wide Web
  • Explicación de conceptos como HTML, HTTP y URL
  • Guía para crear páginas web y configurar servidores
  • Hipervínculos a documentación técnica y ejemplos

Legado y relevancia

  • Inicio de la era digital abierta y descentralizada
  • Base para el desarrollo de navegadores, buscadores y plataformas web
  • Inspiración para iniciativas como el Contract for the Web
  • Restaurada por CERN en 2013 como patrimonio digital

Enlaces operativos

  • info.cern.ch – Página original restaurada
  • Nacimiento de la Web – CERN
  • World Wide Web Consortium (W3C)
  • Web Foundation

Fundada en 1880, la NAD es la principal organización civil en Estados Unidos que defiende los derechos de las personas sordas y promueve el uso y reconocimiento del American Sign Language (ASL) como lengua legítima y culturalmente rica.

Organización responsable

  • Entidad: CERN (European Organization for Nuclear Research)
  • Ubicación: Ginebra, Suiza
  • Propósito: Facilitar el intercambio de información entre científicos mediante hipertexto

Contenido original

  • Descripción del proyecto World Wide Web
  • Explicación de conceptos como HTML, HTTP y URL
  • Guía para crear páginas web y configurar servidores
  • Hipervínculos a documentación técnica y ejemplos

Legado y relevancia

  • Inicio de la era digital abierta y descentralizada
  • Base para el desarrollo de navegadores, buscadores y plataformas web
  • Inspiración para iniciativas como el Contract for the Web
  • Restaurada por CERN en 2013 como patrimonio digital
